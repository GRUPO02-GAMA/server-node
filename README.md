# CRUD com NodeJS

Sistema criado para gestão de agenda

## Funcionalidades

- Adição de um novo contato
- Listagem de todos os contatos da agenda
- Edição de um contato
- Exclusão de um contato

## Stack utilizada

**Front-end:** HTML5, CSS

**Back-end:** JavaScript, NodeJS
